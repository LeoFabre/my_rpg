1. How to Play ?
	You can Launch the game with './my_rpg'.
	You earn XP when you fight.
	The difficulty will depend of your level in-game.

2. Combats :
	Press space when the bar is full to hit with maximal damage !

2. Keys :
	Walk:
		Use [arrows].
	Open Inventory:
		Use [I]
	Open Quest Book:
		Use [Q]
	Interract:
		Use [E]
	Pause:
		Use [Esc]
